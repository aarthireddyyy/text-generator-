# Text Generation API

A REST API that serves Google's **Flan-T5-Small** language model for text-to-text generation, containerized with Docker and deployed on Hugging Face Spaces.

## Tech Stack

`Python` `FastAPI` `HuggingFace Transformers` `PyTorch` `Docker` `Uvicorn`

## What it does

Exposes a simple HTTP endpoint that takes any text prompt and returns a model-generated response using `google/flan-t5-small` — a lightweight instruction-tuned LLM.

## Endpoints

| Method | Route | Description |
|---|---|---|
| GET | `/` | Health check |
| GET | `/generate?text=` | Run inference and return generated text |

**Example:**
```
GET /generate?text=translate English to French: Hello
→ {"output": "Bonjour"}
```

Interactive API docs available at `/docs` (Swagger UI, auto-generated by FastAPI).

## Run Locally

```bash
pip install -r requirements.txt
uvicorn app:app --host 0.0.0.0 --port 7860
```

## Run with Docker

```bash
docker build -t text-gen-api .
docker run -p 7860:7860 text-gen-api
```

## Deployment

Deployed on [Hugging Face Spaces](https://huggingface.co/spaces) using a custom Docker runtime. The container runs as a non-root user following security best practices.
